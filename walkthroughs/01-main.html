01 — The Entry Point

Where the browser starts. Every other file flows from here.

What this file does

Grabs the <canvas> element from the HTML page — this is the rectangle Babylon.js draws into.

Creates a Babylon Engine, which sets up WebGL (the browser's 3D graphics system) and connects it to the canvas.

Creates your Game, which builds the scene, player, camera, and input system.

Starts the render loop — tells the engine to call game.render() roughly 60 times per second, forever.

Listens for window resize so the canvas never looks stretched or blurry.
